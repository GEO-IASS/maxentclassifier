running maxEntClassifier.py on its creates a maximum entropy model that utilizes all possible features
running maxEntClassifier.py [features] ... creates a maximum entropy model with given features
can add flag "-w" at end of command line, to have the program display the weights while updating

no auxiliary data files or modules should be needed. processData.txt and testData.txt are
the training and testing data that the program uses. These are edited to not include instances
with missing entries. adult.data.txt and testing.txt are the original training and testing files
